# Calculatrice Qt
## 📱 Aperçu

Cette calculatrice offre une interface utilisateur moderne et intuitive, permettant d'effectuer des opérations arithmétiques de base.

### Fonctionnalités principales
- ➕ Opérations de base : addition, soustraction, multiplication, division
- 🔢 Gestion des nombres décimaux
- ⌨️ Support du clavier physique
- 🎨 Interface moderne et intuitive

## 🚀 Installation

### Prérequis
- Qt 6.5 ou supérieur
- Compilateur C++17
- CMake 3.16+

### Compilation
```bash
git clone https://github.com/ElPistone/calculatriceQT.git
```
- Ouvrir l'application QT creator
- Ouvrir le fichier calculatrice.pro à partir de QT
- Configurer le projet pour compiler
